# CodeCooperation
## 🎈CodeCooperation 서비스 소개
* 개인 개발자들이 모여 개발 프로젝트를 찾을 수 있어요!
* 팀원이 없어서 문제인가요? 원하는 게시물을 선택하여 참여 가능해요!

## ❓기능 소개

- 본인의 사용 스택을 참고하여 언어별로 고를 수 있어요.
- 같이 하고싶지만 상대가 없을때 프로젝트를 등록하여 여러 사람들과 함께 진행할 수 있어요.
